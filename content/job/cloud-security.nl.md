# Cloud Security Engineer (Azure/AWS)

Ben je een security-minded professional met expertise in cloud security architecturen? Heb je passie voor het beschermen van kritieke infrastructuur tegen threats? Zoek je een rol waar je security by design implementeert? Dan zoeken we jou!

Als Cloud Security Engineer bij DevPort ben je verantwoordelijk voor het ontwerpen, implementeren en beheren van uitgebreide security oplossingen in cloudomgevingen. Je adviseert klanten over security best practices, implementeert IAM-beleid, configureert security monitoring en zorgt voor compliance met regelgeving. Je werkt samen met DevOps engineers, architects en development teams om security in alle lagen van de stack in te bouwen.

Bij DevPort ben je in dienst, maar werk je op detacheringsbasis voor onze klanten. Je krijgt de vrijheid en beloning van ondernemerschap, met de zekerheid van een vast contract.

## Wat we zoeken:

* Minimaal 5 jaar ervaring in cloud security of DevSecOps
* Diepgaande kennis van Azure Security Center, Defender, IAM EN/OF AWS IAM, Security Groups, VPC security
* Expertise in compliance frameworks (SOC2, ISO27001, CIS Benchmarks, NIST)
* Container security en Kubernetes security kennis
* Infrastructure as Code security (Terraform security, policy as code)
* SIEM/monitoring tools expertise (Azure Monitor, CloudTrail, Splunk, ELK)
* Network security en cryptography basics
* Vulnerability management en threat modeling
* Automation en scripting (PowerShell, Python, Bash)
* Bewezen ervaring met cloud security implementaties

## Wat je onderscheidt:

* Certified Information Systems Security Professional (CISSP) of Azure Security Engineer Associate / AWS Certified Security
* CKAD (Certified Kubernetes Application Developer) met security focus
* DevSecOps pipeline expertise
* Zero-trust architectuur implementatie
* Incident response ervaring
* Threat intelligence achtergrond

## Wat we bieden:

* Je wordt gedetacheerd bij uitdagende opdrachten bij onze klanten - vaak met hoge security requirements
* Mogelijkheid om een security-first cultuur te bouwen
* Vast contract met aantrekkelijke arbeidsvoorwaarden
* Budget voor security certificaties (CISSP, Azure/AWS Security)
* Samenwerking met ervaren security professionals
* Autonomie in security design en implementatie
* Budget voor training, conferenties en security research
* Ervaring opdoen met geavanceerde security tools en praktijken
