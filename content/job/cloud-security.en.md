# Cloud Security Engineer (Azure/AWS)

Are you a security-minded professional with expertise in cloud security architectures? Do you have a passion for protecting critical infrastructure against threats? Looking for a role where you implement security by design? Then we're looking for you!

As a Cloud Security Engineer at DevPort, you're responsible for designing, implementing, and managing comprehensive security solutions in cloud environments. You advise clients on security best practices, implement IAM policies, configure security monitoring, and ensure compliance with regulatory requirements. You work with DevOps engineers, architects, and development teams to build security into all layers of the stack.

At DevPort, you're employed by us, but work on secondment for our clients. You get the freedom and rewards of entrepreneurship, with the security of a permanent contract.

## What we're looking for:

* Minimum 5 years of experience in cloud security or DevSecOps
* In-depth knowledge of Azure Security Center, Defender, IAM AND/OR AWS IAM, Security Groups, VPC security
* Expertise in compliance frameworks (SOC2, ISO27001, CIS Benchmarks, NIST)
* Container security and Kubernetes security knowledge
* Infrastructure as Code security (Terraform security, policy as code)
* SIEM/monitoring tools expertise (Azure Monitor, CloudTrail, Splunk, ELK)
* Network security and cryptography basics
* Vulnerability management and threat modeling
* Automation and scripting (PowerShell, Python, Bash)
* Proven experience in cloud security implementations

## What sets you apart:

* Certified Information Systems Security Professional (CISSP) or Azure Security Engineer Associate / AWS Certified Security
* CKAD (Certified Kubernetes Application Developer) with security focus
* DevSecOps pipeline expertise
* Zero-trust architecture implementation
* Incident response experience
* Threat intelligence background

## What we offer:

* You'll be seconded to challenging assignments at our clients - often with high security requirements
* Opportunity to build a security-first culture
* Permanent contract with attractive employment conditions
* Budget for security certifications (CISSP, Azure/AWS Security)
* Collaboration with experienced security professionals
* Autonomy in security design and implementation
* Budget for training, conferences, and security research
* Gain experience with advanced security tools and practices

Are you the Cloud Security Engineer we're looking for? Send us your CV and let's talk!
