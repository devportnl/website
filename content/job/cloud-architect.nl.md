# Cloud Solutions Architect (Azure/AWS)

Ben je een ervaren architect met visie op het ontwerp van schaalbare, veilige en kosteneffectieve cloud-oplossingen? Heb je diepgaande kennis van Azure en AWS? Zoek je een senior rol waar je impactvolle transformaties aanvoert? Dan zoeken we jou!

Als Cloud Solutions Architect bij DevPort adviseer je klanten en ontwerp je complexe cloud-infrastructuur. Je adviseert over cloud strategy, helpt bij de selectie van de juiste services, en designed multi-tier architecturen die aan de huidige en toekomstige behoeften van klanten voldoen. Je werkt samen met DevOps engineers, security specialists en development teams om end-to-end oplossingen op te leveren.

Bij DevPort ben je in dienst, maar werk je op detacheringsbasis voor onze klanten. Je krijgt de vrijheid en beloning van ondernemerschap, met de zekerheid van een vast contract.

## Wat we zoeken:

* Minimaal 7 jaar ervaring in cloudinfrastructuur, waarvan minstens 5 jaar als architect
* Diepgaande kennis van Azure (AKS, App Service, Azure DevOps, Security Center) EN/OF AWS (EKS, EC2, Lambda, IAM)
* Expertise in cloud security architecturen en compliance (SOC2, ISO27001, HIPAA)
* Infrastructure as Code en automation expertise
* Microservices en serverless architectuur kennis
* Cost optimization en cloud financial management
* Networking, storage en database architecture
* Bewezen ervaring met succesvolle cloudmigraties en -transformaties
* Consultancy en communicatievaardigheden op C-level
* Sterke probleemoplossingsvaardigheden

## Wat je onderscheidt:

* Azure Solutions Architect Expert of AWS Solutions Architect Professional certificatie (of hoger)
* DevOps expertise
* Kubernetes kennis
* Security architecture achtergrond
* Ervaring met cloud governance frameworks
* Machine Learning / AI platform kennis
